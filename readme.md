# Materi ReactNative App
Materi reactNative dengan mengunakan [Expo Framework](https://expo.io), untuk mebuat universal React applications namun dengan beberapa [limitasi](https://docs.expo.io/build-reference/limitations/).
Untuk melewati limitasi tersebut kita bisa mengantinya menjadi [native project](https://docs.expo.io/expokit/eject/) lihat pada step 3-6.  

Target:  
- Sebelumnya telah memahami javascript, nodejs, dan npm
- Akan dapat Membuat ui aplikasi dengan react native

## Library Penunjang

### Navigation
Mengunakan library [react-navigation](https://reactnative.dev/docs/navigation#react-navigation) untuk navigasi, silahkan baca dokumentasi lengkap untuk isntalasi lengkap dan pengunaanya.

### UI Element (Supplement)
Untuk memudahkan UI mengunakan [React Native Element](https://reactnativeelements.com/) yang menjamin konsistenti ui antar platform android, ios dan web
